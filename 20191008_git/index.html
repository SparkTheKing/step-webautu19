Система контроля версий.

  • Репозиторий(репа) - хранилище ваших даных(текущий проект)
  • Коммит - точка сохранения(восстановления). По коммитам можно перемещаться и видеть ваш проект в разных стадиях работы
  • Пуш - действие загрузки данных на сервер. (пушануть на гитхаб)
  • Клон - действие создания копии с существующего репозитория на ваш компьютер с помощью адреса (https://github.com/so2niko/step-webautu19.git)(склонировать репу)
  • Ветка(Брэнч) - отдельный экземпляр вашей работы. Возможность без ущерба к основному коду работать и экспериментировать.
